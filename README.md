## 🚀 איך להתחיל?

1. **פתח את הקובץ `index.html` בדפדפן**
   - לחץ לחיצה כפולה על `index.html`
   - או גרור אותו לדפדפן

2. **התאמה אישית**
   - ערוך את קובץ `index.html` כדי להוסיף את המידע שלך
   - שנה צבעים ב-`styles.css` (משתני CSS ב-`:root`)
   - התאם תמונות ופרויקטים

## 🎨 התאמה אישית

### שינוי צבעים

ב-`styles.css`, שנה את המשתנים ב-`:root`:

```css
:root {
    --primary-color: #6366f1;
    --secondary-color: #8b5cf6;
    --accent-color: #ec4899;
    /* ... */
}
```

### הוספת תמונה אישית

החלף את ה-URL בתוך `.profile-pic::before` ב-`styles.css`:

```css
.profile-pic::before {
    background: url('path/to/your/image.jpg') center/cover;
}
```

### עדכון מידע אישי

ערוך את הטקסטים בקובץ `index.html`:
- שם בסקציית Hero
- תיאור בסקציית About
- פרטי קשר בסקציית Contact
- לינקים לרשתות חברתיות

### הוספת פרויקטים

העתק את הקוד של `.project-card` והתאם אותו:

```html
<div class="project-card">
    <div class="project-image">
        <!-- תמונת הפרויקט -->
    </div>
    <div class="project-content">
        <h3>שם הפרויקט</h3>
        <p>תיאור הפרויקט</p>
        <div class="project-tags">
            <span class="tag">טכנולוגיה 1</span>
            <span class="tag">טכנולוגיה 2</span>
        </div>
    </div>
</div>
```
